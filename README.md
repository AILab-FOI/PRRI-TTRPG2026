# PRRI-TTRPG2026

An AI TTRPG game master support system developed by students and the [Artificial Intelligence Laboratory](https://ai.foi.hr/) at the [University of Zagreb Faculty of Organization and Informatics](https://www.foi.unizg.hr/). The game is developed using Python and RenPy. More details available at [itch.io](https://ailab-foi.itch.io/prri-ttrpg2026).
